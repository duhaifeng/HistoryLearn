# HistoryLearn

面向复习与检索的“分地区/国家历史”Markdown 知识库。

## 项目结构

- `China/`：中国历史（以朝代为主线，已补齐近现代至今）
- `Japan/`：日本历史（从绳文到平成/令和）
- `India/`：印度历史（从印度河文明到当代）
- `Europe/`：欧洲历史总览（大叙事口径）
- `Egypt/`：埃及历史（王朝分期）
- `America/`：北美历史（以美国史为主线的大叙事）

每个目录通常包含：

- `00_*总览.md`：时间线 + 分期表 + 演变图 + 速查（入口）
- `01_...md`、`02_...md`…：按时间顺序的分期正文

## 编号与命名规范（约定）

- **编号连续**：同一目录内，文件名前缀使用两位数字（`00`、`01`…），按时间顺序递增。
- **不使用“专题/附录”文件名**：若有补充内容，也应进入编号序列，保持体系可检索与可排序。
- **改名用 `mv`**：需要调整文件名/编号时，用 `mv` 方式改名，以便保留版本历史。

## 推荐阅读顺序

1. 先读对应目录的 `00_*总览.md` 建立时间框架
2. 再按编号顺序阅读分期正文（`01_...` → `02_...` → …）
3. 需要快速回顾时，可优先查看总览中的“速查/演变图”

# 绳文时代（约前14000年—约前4世纪）

## 时代概述

绳文时代是日本列岛最早的定居文化时期，以其标志性的绳纹陶器（表面压印有绳索纹样的陶器）而得名。这一时代的跨度极长，从约前14000年延续到约前4世纪，长达一万余年，是日本历史上持续时间最久的时期。绳文时代的居民以狩猎、采集和渔捞为主要生存方式，过着定居或半定居的生活。与旧大陆同时期的农业革命不同，绳文人在漫长的岁月中发展出了一种独特的、以丰富的自然资源为基础的非农业定居文明。

绳文时代的时间范围通常被划分为六个阶段：草创期（约前14000—前10000年）、早期（约前10000—前5000年）、前期（约前5000—前3500年）、中期（约前3500—前2500年）、后期（约前2500—前1300年）和晚期（约前1300—前4世纪）。各阶段在陶器风格、聚落规模、生存策略和社会组织上都有显著差异。绳文时代的文化成就令人瞩目：其陶器是世界上已知最古老的陶器之一，造型丰富多变；土偶造型神秘而富于想象力；漆器工艺的历史可追溯至约九千年前；大规模聚落（如三内丸山遗迹）显示出相当成熟的社会组织能力。

日本列岛在绳文时代初期因冰期结束、海平面上升而逐渐与亚洲大陆分离，形成了相对封闭的海洋性岛屿环境。这一地理特征深刻影响了绳文文化的独立发展——绳文人充分利用了列岛丰富的海洋资源、落叶阔叶林的坚果以及山地的猎物，发展出了在世界史上独树一帜的狩猎采集型定居文明。绳文时代虽然没有文字记录，但大量的考古发掘为我们还原了这一漫长时期的面貌，使其成为日本历史和文化认同的重要源头。

## 社会组织与生活形态

绳文时代尚未形成天皇制度或任何形式的统一政权，其社会组织以血缘关系为纽带的部落和村落为基本单位。绳文社会的特征可概括如下：

| 方面 | 特征 |
|------|------|
| 基本社会单位 | 由数户至数十户构成的村落，居民之间多有血缘关系 |
| 住居形式 | 竖穴住居（在地面挖出浅坑、立柱搭建茅草屋顶的半地穴式房屋） |
| 聚落布局 | 住居围绕中央广场（可能用于祭祀或集会）呈环形或马蹄形分布 |
| 生存方式 | 狩猎（鹿、野猪等）、采集（栗子、橡子、核桃等坚果）、渔捞（鱼、贝类） |
| 社会分化 | 总体上平等的社会，无明显阶层分化（晚期可能出现初步分化） |
| 贸易交换 | 存在远距离物资交换（如黑曜石、翡翠、沥青等在数百公里范围内流通） |
| 人口规模 | 中期鼎盛时期日本列岛总人口估计约26万人，东日本（东北和关东地区）人口密度较高 |
| 丧葬习俗 | 屈葬（将遗体蜷曲埋葬）为主，部分地区有拔齿习俗（可能与成人仪式有关） |

绳文社会被认为是一个相对和平、平等的社会。考古发掘中较少发现因暴力致死的人骨证据，聚落也缺乏防御设施，这与后来弥生时代环濠聚落的出现形成鲜明对比。绳文人的精神世界丰富而复杂——土偶、石棒、石冠等祭祀用具大量出土，显示出对自然力量和生命循环的深厚敬畏。

## 重大历史事件

### 一、日本列岛与大陆的分离（约前14000—前10000年）

约两万年前的最后一次冰期极盛期（末次冰期），全球海平面比现在低约120米以上。当时的日本列岛与亚洲大陆之间存在陆桥连接——北海道通过库页岛与西伯利亚相连，九州通过朝鲜海峡与朝鲜半岛相连。大型动物（猛犸象、纳玛象、大角鹿等）和旧石器时代的人类正是通过这些陆桥迁徙到日本列岛的。

约前14000年起，随着全球气候变暖，冰川融化，海平面逐渐上升。到约前10000年前后，日本列岛与大陆之间的陆桥被海水淹没，日本海形成，日本列岛成为被海洋环绕的群岛。这一地理变化对日本文化的发展具有深远意义——列岛的相对封闭性使绳文文化得以在较少外部干扰的情况下独立发展数千年，形成了与大陆文化截然不同的特色。

与此同时，气候变暖带来了植被的巨变。原本覆盖列岛的针叶林逐渐被落叶阔叶林（栗树、橡树、核桃等）和常绿阔叶林（照叶树林）所取代，海洋中的鱼类和贝类资源也因暖流作用而更加丰富。这种温暖湿润的自然环境为绳文人提供了充足的食物来源，使他们能够在不从事农业的情况下实现长期定居。

### 二、绳文陶器的诞生与演变（约前14000—前4世纪）

绳文陶器是绳文时代最具标志性的文化遗产，也是世界上已知最古老的陶器之一。最早的绳文陶器出土于青森县大�的平遗迹等地，年代约为前13000年，器形简单，为尖底或圆底的深钵，外壁有绳纹装饰或无纹。这些早期陶器的制作与使用，标志着日本列岛居民从旧石器时代的游动生活向新石器时代的定居生活过渡。

绳文陶器在漫长的岁月中经历了丰富的风格演变：

- **草创期—早期**（前14000—前5000年）：器形以尖底深钵为主，纹饰简朴，以绳纹、贝壳压纹为常见装饰
- **前期**（前5000—前3500年）：器形多样化，出现平底器、浅钵和壶形器；纹饰开始复杂化
- **中期**（前3500—前2500年）：绳文陶器的艺术巅峰。以信浓川流域（今新潟县—长野县一带）出土的"火焰型土器"最为著名——这些陶器口沿上装饰着高耸的火焰状突起，造型奔放夸张、充满动感和想象力，堪称史前艺术的杰作。中期陶器的器壁变厚，体量增大，装饰繁复
- **后期—晚期**（前2500—前4世纪）：陶器风格趋于简洁精致，器壁变薄，出现了注口土器（带注水口的壶形器）等新器形。纹饰回归朴素，表面打磨光滑

绳文陶器的制作采用手工捏塑和盘泥条法，不使用陶轮，在露天篝火中烧成（温度约800—900°C）。虽然制作技术相对原始，但绳文陶器在造型和装饰上展现出的丰富想象力和审美追求，在世界史前陶器中独树一帜。

### 三、贝冢文化的形成与发展（约前7000—前1000年）

贝冢是绳文时代居民丢弃贝壳和其他食物残渣形成的遗迹堆积物，是研究绳文社会最重要的考古资料之一。日本全国已发现约2500处贝冢遗迹，主要分布在关东地区的太平洋沿岸，其中东京湾周边尤为集中。

贝冢中保存了大量的贝壳、鱼骨、兽骨、鸟骨以及人工制品（陶器、骨角器、石器等），由于贝壳中的�ite成分使土壤呈碱性，有利于有机物质的保存，因此贝冢中甚至能发现保存完好的人骨——这为研究绳文人的体质特征、饮食结构和疾病状况提供了珍贵资料。通过分析贝冢中的贝壳种类（蛤、牡蛎、蚬等），学者们可以推断当时的海岸线位置、水温变化和季节性资源利用模式。

关东地区最著名的贝冢之一是千叶县的加曻利贝冢，该遗迹规模巨大，出土了丰富的陶器、石器和人骨，被指定为国家特别史迹。贝冢文化的繁荣反映了绳文人对沿海环境的高度适应和对海洋资源的系统性利用。

### 四、大规模聚落的形成——三内丸山遗迹（约前3900—前2200年）

三内丸山遗迹位于青森县青森市，是绳文时代中期最具代表性的大规模聚落遗址，也是理解绳文社会组织水平的关键证据。该遗迹于1992年开始发掘调查，出土成果震动了日本考古学界，极大地改变了人们对绳文时代的认识。

三内丸山遗迹的主要发现包括：

- **聚落规模**：遗迹面积约42公顷，持续使用约1700年（前3900—前2200年），是已知绳文时代最大的聚落之一
- **大型掘立柱建筑**：发现了六根直径约1米的巨大栗木柱坑遗迹，柱间距统一为4.2米（约14尺），显示出精确的设计和规划能力。这一建筑的用途尚存争议——可能是瞭望塔、祭祀设施或集会场所
- **竖穴住居**：发掘出约800处以上的竖穴住居遗迹，按用途分为居住区和墓葬区
- **大型竖穴住居**：发现了长32米、宽10米的超大型竖穴建筑，可能用于集会或共同作业
- **栗林管理**：遗迹周围发现了大量栗树花粉和栗木遗存，DNA分析显示这些栗树的基因多样性较低，表明绳文人可能有意识地选育和管理栗林——这是一种介于采集和农业之间的"半栽培"行为
- **远距离贸易**：出土了产自北海道的黑曜石、新潟的翡翠、岩手的琥珀以及沥青等来自远方的物资，显示三内丸山聚落参与了广泛的物资交换网络

三内丸山遗迹的发现证明，绳文社会并非此前学界所认为的那样简单原始——至少在某些地区和时期，绳文人已经发展出了相当复杂的社会组织和生存策略。该遗迹于2000年被指定为国家特别史迹，2021年作为"北海道·北东北绳文遗迹群"之一被列入联合国教科文组织世界文化遗产。

### 五、绳文时代的衰落与弥生文化的传入（约前10世纪—前4世纪）

绳文时代的晚期（约前1300—前4世纪），日本列岛经历了一次显著的气候冷化。气温下降导致落叶阔叶林（尤其是栗树）的分布范围缩小，食物资源减少，东日本地区的绳文人口明显下降。一些学者估计，晚期的日本列岛总人口可能下降到约7.6万人，仅为中期鼎盛时的三分之一左右。

与此同时，约从前10世纪开始（有争议，部分学者主张更早），朝鲜半岛的水稻种植技术和大陆系统的磨制石器、新式陶器逐渐传入九州北部地区。最初这些大陆要素与绳文文化并存（所谓"绳文晚期弥生化"或"菜畑遗迹"阶段），但到约前4世纪，以水稻种植为核心、伴随金属器使用的弥生文化已经在九州北部确立，并开始向东扩展。

绳文时代向弥生时代的过渡并非一夜之间的突变，而是一个在不同地区以不同速度进行的渐进过程。九州北部最早进入弥生时代，而东北地区直到前1世纪左右才完成过渡；北海道则从未进入弥生文化圈，而是发展出了独立的"续绳文文化"，后来演变为阿伊努文化。关于弥生文化的传入方式，学界存在"渡来人替代说"（大量大陆移民取代了原住的绳文人）和"文化传播说"（主要是技术和文化的传播而非人口替代）两种主要观点，近年的遗传学研究倾向于支持两种因素并存的混合模型。

## 文化与成就

### 绳文陶器

绳文陶器是绳文时代最突出的文化遗产。"绳文"之名本身即来源于陶器表面以绳索压印出的纹饰。绳文陶器的年代可追溯至约前14000年，与中国长江流域的早期陶器共同构成了世界上已知最古老的陶器传统。

绳文陶器最引人注目的特征是其充满想象力的造型和装饰。中期（前3500—前2500年）的"火焰型土器"尤为壮观——巨大的口沿突起如同燃烧的火焰或涌动的波浪，整个器身布满繁复的立体装饰，造型之奔放在世界史前陶器中罕有其匹。冈本太郎（日本著名现代艺术家）1951年在东京国立博物馆初见绳文陶器时为之震撼，盛赞其"四次元的艺术"，由此在战后日本掀起了"绳文美学"的再发现热潮。

绳文陶器虽然在技术上不使用陶轮、烧成温度较低，但其多样性和创造性令人惊叹——从实用的炊煮器到精美的祭祀用器，从朴素无纹的素面陶到装饰繁复的火焰型土器，绳文陶器展现了史前日本人丰富的审美意识和精神世界。

### 土偶

土偶是绳文时代用粘土制作的小型人形或动物形塑像，全日本已出土约两万件。土偶的造型多为女性形象，身体特征（乳房、臀部、腹部）被夸张表现，四肢和面部则相对简略。多数土偶被有意打碎后分散埋藏，完整出土者极少。

土偶的用途至今仍存争议，主要假说包括：用于祈求丰产和安产的祭祀仪式、治病仪式中的替身（将疾病转移至土偶后打碎）、死者的殉葬品等。无论其具体功能为何，土偶无疑反映了绳文人丰富的精神信仰和象征思维。

最著名的土偶作品包括：
- **"遮光器土偶"**（青森县龟冈遗迹出土，晚期）：因其巨大的眼部造型酷似因纽特人的雪地护目镜而得名，造型奇异，是绳文文化最具辨识度的象征之一，被指定为国宝
- **"合掌土偶"**（青森县风张遗迹出土，后期）：呈双手合十的坐姿，表情安详，被指定为国宝
- **"缩面土偶"**（长野县棚畑遗迹出土，中期）：呈现明显怀孕的女性形象，被视为"地母神"信仰的证据
- **"中空土偶"**（北海道著保内野遗迹出土，后期）：内部中空的大型坐姿土偶，被指定为国宝

### 漆器工艺

绳文时代的漆器工艺是一项令人惊讶的发现。2000年在北海道函馆市�的垣ノ島遗迹出土了年代约前7000年的漆器碎片（红漆涂装的容器），这是目前世界上已知最早的漆器遗物之一。绳文人从漆树（ウルシ）中采集树液，用于涂饰木器、陶器和编织物，颜色以红漆和黑漆为主。

�的内丸山遗迹也出土了大量绳文中期的漆器残片，包括漆涂的梳子、耳饰和容器等。这些发现证明，绳文人已经掌握了相当成熟的漆器制作技术——包括漆液的采集、精制、调色和涂装——这一传统后来成为日本文化中"漆艺"（うるし）的源流，并在此后数千年中发展成为日本最具代表性的工艺美术之一。

### 竖穴住居

竖穴住居是绳文时代最典型的居住形式。其建造方法是在地面挖出直径3—7米、深约30—80厘米的圆形或方形浅坑，在坑底中央设置炉灶，四周竖立支柱，搭建茅草、树皮或泥土覆盖的屋顶。这种半地穴式的结构具有冬暖夏凉的优点，适应了日本列岛四季分明的气候。

竖穴住居通常呈环形排列，围绕中央的广场（用于祭祀、集会或墓葬），形成有组织的村落空间布局。一个典型的绳文村落由5—20栋竖穴住居组成，居住人口约30—100人。住居的使用寿命约10—20年，废弃后会在附近重建。

### 石器与骨角器

绳文时代的石器技术继承了旧石器时代的传统，并在此基础上发展出了磨制石器——包括石斧（用于伐木和加工木材）、石皿和磨石（用于碾碎坚果）、石镞（箭头，用于狩猎）、石锤等。此外，绳文人还广泛使用骨角器（鹿角制的鱼钩、刺突具、缝纫针、簪等）和贝器。

黑曜石是绳文时代最重要的石材之一。它主要产于北海道十�的田、长野县和田�的、东京都神津岛等少数火山岛产地，但在远离产地数百公里的遗迹中也有大量出土，这证明了绳文时代已存在广泛的远距离物资交换网络。翡翠（主要产自新潟县糸鱼川）也被加工成勾玉等装饰品，在广大地区流通。

### 精神世界与祭祀

绳文人拥有丰富而复杂的精神世界。除了前述的土偶之外，以下考古发现也反映了他们的信仰和仪式：

- **石棒与石冠**：可能与男性生殖力崇拜有关的石制祭祀具
- **环状列石**（ストーンサークル）：大汤环状列石（秋田县鹿角市）等遗迹中发现的由石块排列成圆形的大型祭祀设施，可能与太阳历或祖先祭祀有关
- **拔齿习俗**：绳文中期以后广泛存在的将健康门牙拔除的习俗，可能与成年仪式、婚姻或丧葬有关
- **屈葬**：将遗体蜷曲成胎儿姿势埋葬的丧葬方式，有学者认为这象征着回归母胎、期待再生

这些遗存表明，绳文人虽然没有文字，但已经发展出了相当复杂的宗教观念和仪式实践，其核心可能围绕着对自然力量的敬畏、对生命循环的信仰以及对祖先的祭祀。

## 重要遗迹

### 三内丸山遗迹（青森县青森市）

日本绳文时代中期最大规模的聚落遗址，存续约1700年（前3900—前2200年）。出土了大型掘立柱建筑遗迹、超大型竖穴建筑、约800处竖穴住居及大量陶器、石器、土偶和来自远方的翡翠、黑曜石等物资。2000年被指定为国家特别史迹，2021年列入世界文化遗产。

### 大平山元I遗迹（青森县外ヶ浜町）

出土了年代约前14500年的陶器碎片，是世界上已知最古老的陶器出土地之一。这些陶器的发现对于理解陶器的起源和旧石器时代向新石器时代的过渡具有重大学术意义。

### 加曻利贝冢（千叶县千叶市）

关东地区最具代表性的大型贝冢遗迹之一，由南北两个环状贝冢组成，年代跨绳文中期至后期。出土了大量陶器（"加曻利E式""加曻利B式"成为绳文陶器编年的标准型式）、石器、骨角器和人骨。被指定为国家特别史迹。

### 大汤环状列石（秋田县鹿角市）

绳文后期（约前2000—前1500年）的大型环状列石遗迹，由"万座环状列石"和"野中堂环状列石"两个相距约90米的石圈组成。这些石圈直径达40—50米，由大量河卵石精心排列而成，被认为是绳文时代重要的祭祀·墓葬设施。2021年列入世界文化遗产。

### 垣ノ島遗迹（北海道函馆市）

绳文早期至中期的大型聚落遗址，出土了年代约前7000年的漆器——世界上已知最古老的漆器遗物之一。遗迹还发现了大规模的"盛土"（人工堆积的土丘，可能用于祭祀或废弃物处理），以及精美的足形付土版（模拟婴儿脚印的粘土板，可能用于纪念夭折的婴儿）。2021年列入世界文化遗产。

## 衰亡与影响

### 绳文时代的终结

绳文时代的终结不是一次突发事件，而是一个持续数百年的渐进过程。约前10世纪起，来自朝鲜半岛的水稻种植技术开始在九州北部出现。这种新的生存方式——以灌溉水田农业为核心、辅以金属器使用——代表了一种与绳文文化根本不同的社会经济模式。到约前4世纪，九州北部和西日本已基本完成向弥生文化的过渡。然而，东日本的绳文传统延续时间更长，北海道则始终未接受弥生文化，而是发展出了独立的"续绳文文化"。

绳文文化终结的原因是多方面的：气候冷化导致食物资源减少、绳文人口下降；来自大陆的稻作农业提供了更为稳定和丰富的食物来源；随农业一同传入的金属器和新社会组织形式对绳文社会形成了冲击。但值得注意的是，这一过渡并非简单的文化替代——绳文文化的许多元素（如陶器传统、特定的精神信仰、与自然和谐共处的生活态度）在弥生时代乃至后世的日本文化中继续传承。

### 对后世的影响

1. **日本文化的基层**：尽管弥生时代的大陆文化对日本后世发展影响巨大，但绳文时代留下的文化基因并未消失。日本独特的自然观（万物有灵的思想，与神道教的渊源）、对森林和海洋的亲近感、某些饮食习惯（如食用坚果和海产品的传统）都被认为可以追溯到绳文时代。

2. **遗传学遗产**：现代日本人的基因构成是绳文系（源自绳文时代居民）和弥生系（源自大陆渡来人）两大成分的混合。遗传学研究表明，绳文系基因在现代日本人中的比例因地区而异——阿伊努人和琉球人中绳文系成分较高，本州日本人中弥生系成分较高。绳文人的遗传特征（如较深的眼窝、较多的体毛、双眼皮、湿型耳垢等）在部分现代日本人中仍可观察到。

3. **考古学与国民认同**：绳文文化在现代日本的文化认同中占有重要地位。三内丸山遗迹的发掘在1990年代引发了"绳文热"，绳文陶器和土偶成为日本文化的象征符号。2021年"北海道·北东北绳文遗迹群"列入世界文化遗产，进一步强化了绳文时代在日本乃至世界文化遗产中的地位。

4. **对世界史前史研究的贡献**：绳文时代证明了狩猎采集社会也可以实现长期定居、发展出复杂的社会组织和丰富的物质文化——这一认识挑战了"农业革命是定居文明前提"的传统历史叙事，为全球史前史研究提供了重要的比较案例。

---

> **学术备注**：绳文时代的断代长期依赖陶器编年，近年来放射性碳定年法（特别是AMS法）的广泛应用使其绝对年代得到了大幅修正。关于绳文时代的起始年代，学界曾长期使用"约前10000年"的说法，近年来根据最新的碳十四数据已普遍上调至"约前14000年"甚至更早。关于绳文社会的性质（平等社会还是存在阶层分化、单纯的采集经济还是存在初步的植物栽培），学界仍存在活跃的争论。本文采用日本考古学界的主流观点，但读者应注意这些问题并非定论。
